  </Step>
  <Step title="Activez le parrainage">
    Cliquez sur le bouton \*\*Activer le parrainage \*\*pour activer votre code

      ![Parrainage](/images/parrainage.png)
    </Frame>
  </Step>
  <Step title="Votre code de parrainage">
    Votre **code** ainsi que votre **lien de parrainage** apparaitra

    <Frame>
      ![Parrainage 1](/images/parrainage-1.png)
    </Frame>
    <Info>
      Vous pouvez partager votre code n'importe où : sur les réseaux sociaux, sur les serveurs Discord ou dans les descriptions de vos vidéos.
    </Info>
  </Step>
</Steps>

## Comment débloquer les récompenses

Chaque fois que quelqu'un utilise votre code de parrainage, vous recevrez des récompenses bonus.\
Ces bonus sont automatiquement suivis et affichés dans votre tableau de bord.
